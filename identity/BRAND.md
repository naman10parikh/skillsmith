# skillsmith — BRAND

## Name rationale

**skillsmith** = *skill* + *smith* (a maker who forges raw material into a finished tool). The
CLI forges raw intent into a well-formed Claude Code **skill** — scaffolded, validated, and
ready to share. The name is lowercase to match its npm package and CLI binary (`skillsmith`).

## Tagline

**Create, test, and share Claude Code skills.** The npm registry for AI-agent skills.

## Positioning

The toolchain for skill authors: `init` scaffolds with best practices, `test` validates
against the spec, `publish` / `install` / `list` move skills through a GitHub-based registry.
A skill is just a `SKILL.md` markdown file — skillsmith makes writing a good one repeatable.

## Voice

Direct, practical, developer-first. Show the command, then the result. No hype, no filler.

## Landing copy (seed)

> Claude Code skills are markdown files that teach Claude new capabilities. Skillsmith gives
> you the toolchain: scaffold with best practices, validate against the spec, and publish to a
> shared registry. `npx skillsmith init my-skill` to start.

## Marks

- Wordmark: `skillsmith`, lowercase, monospace-friendly.
- Forged from the Energy platform — the anvil/forge motif fits the "smith" identity.
